# Chainpay Project

Chainpay is a platform for businesses to manage cryptocurrency payments and conversions. This platform allows businesses to accept payments in Polygon (POL) or Solana (SOL) and convert to USDC for easy management.

## Features:
- Accept crypto payments in POL or SOL
- Automatic conversion to USDC (Polygon or Solana)
- Payment links and invoicing for businesses
- CEX integrations for easy off-ramping

## Setup:
1. Clone the repository.
2. Install dependencies (if any).
3. Open `index.html` in your browser to view the project.

## Technologies Used:
- HTML, CSS, JavaScript
- Polygon, Solana Blockchains
